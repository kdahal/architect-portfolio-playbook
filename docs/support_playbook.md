# Support Playbook (summary)

- **Primary goals:** Rapid resolution, provide clear escalation paths, gather product feedback from tickets.
- **Escalation tiers**
  1. Level 1: Triage & reproducer creation (Applied Eng)
  2. Level 2: Deep debugging & temporary mitigations (SRE/Platform)
  3. Level 3: Product engineering & design changes
- **Data points to collect:** logs, request IDs, timestamps, environment, API payloads, error traces.
- **When to escalate:** production outage, security impact, data loss, unresolved in 4 business hours.
